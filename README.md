# Introducción PII 2026

Sitio web de ejercicios de reactivación diagnóstica para **Programación Avanzada / Programación II 2026**.

La página fue pensada para que los estudiantes puedan repasar y reactivar conocimientos trabajados en 1.º año de programación en Java, antes de comenzar el curso con mayor profundidad.

## Contenido

El sitio incluye ejercicios organizados en cuatro categorías:

- **Interpretar código**
- **Corregir programas**
- **Completar fragmentos**
- **Crear programas breves**

También están clasificados por nivel:

- **Mínimo viable**
- **Estándar**
- **Desafío**

## Objetivo didáctico

Este material busca:

- reactivar recuerdos de programación de primer año;
- ofrecer una práctica inicial accesible;
- servir como insumo para una evaluación diagnóstica breve en clase;
- favorecer la lectura de código, la corrección de errores y la producción de soluciones simples en Java.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript vanilla
- GitHub Pages
